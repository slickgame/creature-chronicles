# Ranch Systems Art Checklist

This checklist tracks the UI icons and future event art needed for the ranch systems added from Ranch Chores onward. Keep these as placeholder-friendly until the mechanics are stable.

## Highest-value UI icons

| Asset | Purpose | Priority | Notes |
|---|---|---:|---|
| Ranch Materials icon | Resource counter, Ranch Office upgrade/repair costs | High | Should read clearly at 32–64 px. Crate, planks, nails, rope, or bundled supplies. |
| Ranch Damage icon | Condition/damage meter, Morning Report | High | Cracked fence, splintered plank, claw mark, or warning badge. |
| Repair Ranch icon | Manual Repair button | High | Hammer + plank + sparkle/check mark. |
| Security Report icon | Morning Report section | Medium | Lantern, shield, paw tracks, or fence watch. |
| Upkeep Report icon | Field Hauling / repair section | Medium | Cart, toolbox, rope, or supplies. |
| Feed Stock icon | Feed stock and daily feed projection | Medium | Feed sack, hay bale, grain bag. |
| Breeding Comfort icon | Comfort Care and Breeding Pen bonus | Medium | Blanket, heart, calming aura, nesting pillow. |

## Ranch condition badge set

| Badge | Damage Range | Gameplay Meaning | Priority |
|---|---:|---|---:|
| Good | 0–19 | No penalty | High |
| Worn | 20–49 | -5% sleep recovery | High |
| Damaged | 50–79 | -15% recovery, -1 affection | High |
| Critical | 80–100 | -25% recovery, -2 affection | High |

Recommended direction: use the same base ranch/fence silhouette with escalating cracks, warning color, and debris.

## Style notes

- Prioritize readable silhouettes over detail for icons.
- Use a consistent square icon frame so assets can swap into existing UI quickly.
- Keep event illustrations separate from tiny UI icons.
- Do not generate full scene art until the ranch loop is stable.
- First generated batch should probably be: Materials, Damage, Repair, Security, Upkeep, Feed, Breeding Comfort, and the four condition badges.
